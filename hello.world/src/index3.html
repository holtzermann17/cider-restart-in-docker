The page is also updated when the Clojurescript code changes.

Specifically, the reload function inside src/hello/world/frontend/main.cljs is registered to run then.

Changes to src/index.html don't trigger code reloading, so the browser page needs to be reloaded to see changes there.

What about changes to the Clojure code, or the EDN file defining the project?

Those changes be brought into effect with (reset) on the REPL.

Or you can run M-x cider-ns-refresh, which wraps the same functionality.
